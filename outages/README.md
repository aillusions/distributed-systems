# Reproduce a famous outage

**Goal:** take a real postmortem (Cloudflare / AWS / GitHub / …) and recreate the
failure mode in miniature.

**Teaches:** failure modes you'd never have imagined — cascading failure, retry
storms, metastable states, config-push blast radius.

**Per outage**
- A short writeup: the original incident + the mechanism
- A minimal repro (scripts / compose) that triggers the same failure
- What would have prevented or contained it
- Teach-back write-up → site
